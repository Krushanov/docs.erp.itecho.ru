`https://beta.back.erp.itecho.ru/api/departments`

}

```json
{
	"department": {
		"id": 6,
		"name": "IT",
		"created_at": "2023-11-13T13:22:07.000000Z"
	}
}

- `created_at` - дата и время добавления в систему
